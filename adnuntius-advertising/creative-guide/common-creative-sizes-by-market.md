# Common Creative Sizes by Market

This page provides an overview of the most commonly requested ad sizes specified per market. This is intended to simplify the workflow for agencies and production teams when creating assets, especially for cross-border campaigns.

Ô∏è About the Data

* Sorted by Volume: The formats below are listed in descending order based on the number of bid requests (highest to lowest).
* Availability: Please note that not all formats are available across all publishers.
* Reference Only: These lists are based on historical traffic data. They are intended as a technical reference to help identify common specs, not as a strategic recommendation from Adnuntius.
